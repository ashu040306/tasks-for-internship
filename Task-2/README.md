# Task 2: Finding Products with Largest Decrease in Sales

## Objective
Identify the products that experienced the most significant decrease in sales between 2021 and 2022.

## Tools Used
- Python
- Pandas
- SQLite
- SQL
- Google Looker Studio
- VS Code

## Dataset Fields Used
- sku_name
- order_date
- qty_ordered

## Steps Performed

1. Created SQLite database
2. Imported order and product datasets
3. Aggregated sales by product for:
   - 2021
   - 2022
4. Calculated sales difference
5. Identified top 10 products with largest decrease
6. Exported results to CSV
7. Created visualization in Google Looker Studio

## Output Files

- main.py
- sales_2021_by_product.csv
- sales_2022_by_product.csv
- sales_difference_2021_2022.csv
- top_10_products_largest_decrease.csv
